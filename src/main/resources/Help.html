MapAlgos Help


File communication

  • To open map from file either use Load saved map button in main menu or open button in File section in manager
  • To save map use save or save as buttons from File section in manager and choose directory and file name


Creating maps

  • To add new vertex choose vertex from New section and click in the place you want new vertex to be, you can always change this place by dragging vertex somewhere else
  • To add new edge choose edge from New section and check both vertices it should connect, if it should be weighted choose weighted egde and after previous steps write weight in pop-up (only integers allowed)


Visualise algorithms

In submenu Algorithms you can choose algorithm to be visualized and click somewhere in the panel (in case of dfs, bfs, dijkstra or prim click in the starting vertex)


Elements and informations

Use subsection Elements to find certain subgraphs, and Informations to get basic characteristics of given graph/map


Settings

  • Animation speed stands for time to pause when visualize algorithms (milisec)
  • Label visible sets if labels of vertices should be visible
